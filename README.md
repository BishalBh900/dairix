🥛 Dairix – Dairy Management System

Dairix is a Django-based Dairy Management System designed to manage milk collection, farmers, sales, and dairy operations in a simple and efficient way.

🚀 Features
👨‍🌾 Farmer management
🥛 Milk collection tracking
📊 Reports and summaries
🔐 Authentication & user roles (RBAC)
🌐 REST API support using Django REST Framework
📦 Database integration with PostgreSQL / SQL
🛠️ Tech Stack
Python (Django)
Django REST Framework
PostgreSQL / SQLite
HTML, CSS, JavaScript
